# Ambient audio

The audio toggle in the HUD plays a single looping ambient space drone from this
folder. Add the track here with these exact filenames:

- `ambient.ogg` — primary (Ogg Vorbis)
- `ambient.mp3` — fallback (MP3)

Requirements:

- ~2–3 minutes, seamless loop, ≤ 1.5 MB.
- Sourced from the **NASA Audio Collection** (public domain) or **Freesound.org**
  (CC0). Record the source URL and license below.

> The files are intentionally not committed. Until they are added, the audio
> toggle is fully functional but silent (playback fails gracefully).
